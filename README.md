# Oscar Best Director Dataset (1930–2019)

This repository contains data and metadata submitted for a course assignment on data management, metadata creation, and digital preservation. The dataset includes information about Oscar-nominated and Oscar-winning film directors from 1930 to 2019, including gender, race, award status, and film titles.

---

## 📁 Repository Contents

### **1. Data Files (Original Source Data)**
These files represent the original dataset used in this assignment:

- **Oscar Winners - Director copy XML.xlsx**  
- **Oscar Winners - Director copy PDF.pdf**

This dataset was originally obtained from Kaggle:  
**Oscar-Winning Directors Analysis** by *The Devastator / Priyanka Dobhal*  
https://www.kaggle.com/datasets/thedevastator/oscar-winning-directors-analysis  
[1](https://www.kaggle.com/datasets/thedevastator/oscar-winning-directors-analysis)

The Kaggle dataset includes information on:
- Award year  
- Gender  
- Race  
- Director(s)  
- Film title  
- Nomination/Winner status  

The dataset spans nearly nine decades (1930–2019). [1](https://www.kaggle.com/datasets/thedevastator/oscar-winning-directors-analysis)

## 📊 Dataset Description

This dataset documents winners and nominees for the **Academy Award for Best Director**, including:

- Year  
- Gender  
- Race  
- Director(s)  
- Film  
- Nomination/Winner  

The dataset highlights long-term patterns in race and gender representation among Oscar-recognized directors. [1](https://www.kaggle.com/datasets/thedevastator/oscar-winning-directors-analysis)

## 🛡 Long-Term Preservation Considerations

- `.xlsx` is a proprietary Microsoft format; long-term risk of obsolescence.  
- `.pdf` is stable and highly supported.  
- `.xml` (DataCite metadata) is an open standard suitable for preservation.  
- No unusual software requirements beyond:
  - Excel/Google Sheets/LibreOffice  
  - Any PDF reader  
  - Any text editor for XML
